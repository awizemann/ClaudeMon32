## Description

Device gets a Preferences/NVS store and a `set_wifi` serial command ({ssid, pass}) that persists WiFi creds and (re)connects. Keeps secrets out of source. On boot, if creds exist, join WiFi and log the acquired IP. Foundation for P4.1+ (and later NVS token storage). Firmware only.
