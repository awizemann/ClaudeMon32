## Description

Only the daemon (com.claudemon.agent) auto-starts; the web admin is on-demand, so claudemon-admin.local dies on Mac reboot until the user runs `claudemon admin`. For the "just open the URL anytime" promise to hold, add a second launchd agent (e.g. com.claudemon.admin) that runs `claudemon admin`, or fold an optional admin thread into the daemon. Decide one; keep the mDNS advertisement (admin.py) working under it. Low priority — the display works without it; this is about the config console always being reachable by name.
